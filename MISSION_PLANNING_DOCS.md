## 1. БЫСТРЫЙ СТАРТ

### Copter / Plane
1. Выбрать тип ЛА в правой панели (Copter или Plane)
2. Дождаться GPS → HOME появится автоматически
3. Кликнуть левой кнопкой на карте → добавить вейпоинты
4. Двойной клик по вейпоинту → настроить параметры
5. Нажать **"Активировать миссию"** в правой панели
6. Управлять полётом кнопкой **"⏸ Заморозить миссию"**

### QuadPlane (VTOL)
1. Выбрать **QuadPlane** в типе ЛА
2. Дождаться GPS → HOME появится автоматически
3. Кликать на карте → добавить вейпоинты маршрута
4. Точки **S** (Старт) и **L** (Посадка) появятся автоматически
5. При необходимости — перетащить S и L мышкой
6. Двойной клик по S или L → настроить параметры
7. Нажать **"Активировать миссию"**
8. Дрон: взлёт → FW → кружение на S → ожидание → маршрут → L → MC → посадка

## 2. ТИПЫ ЛА И МИССИЙ

| Тип ЛА | Маршрут | Метод загрузки |
|---------|---------|----------------|
| **Copter** | HOME → WP1 → WP2 → ... → RTL | `UploadPlannedMission()` |
| **Plane** | HOME → WP1 → WP2 → ... → RTL | `UploadPlannedMission()` |
| **QuadPlane** | HOME → ВЗЛЁТ → FW → S → WP1..N → L → MC → ПОСАДКА | `UploadVtolMission()` |

Тип ЛА влияет на:
- Доступные режимы полёта (правая панель)
- Минимальный радиус кружения: **80м** для QuadPlane, **5м** для Copter
- Наличие точек S и L (только QuadPlane)
- Формат MAVLink миссии

## 3. ПЛАНИРОВКА МАРШРУТА

### Добавление вейпоинтов
- **Левый клик** на карте → новый WP в конец маршрута
- **Перетаскивание** маркера → изменить позицию WP
- **Правый клик** по маркеру → удалить WP
- **Одинарный клик** по маркеру → показать/скрыть круг радиуса
- **Двойной клик** по маркеру → диалог редактирования

### Диалог редактирования WP
| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| **Высота** | Высота полёта (м) | 100 |
| **Радиус** | Радиус кружения (м) | 20 (Copter), 80+ (QuadPlane) |
| **AutoNext** | Автопродолжение | true |
| **Задержка** | Пауза на точке (сек) | 0 |
| **Витки** | Кол-во кругов (LOITER_TURNS) | 1 |
| **Направление** | CW (по часовой) / CCW (против) | CW |

### Поведение AutoNext
| AutoNext | Витки | Команда | Поведение дрона |
|----------|-------|---------|-----------------|
| ON | > 0 | LOITER_TURNS | N кругов → летит дальше |
| ON | = 0 | NAV_WAYPOINT | Пролетает → дальше |
| OFF | любые | LOITER_UNLIM | Кружит бесконечно → ждёт оператора |

### Изменение радиуса
- Кликнуть по WP → появится круг радиуса + белая точка (resize handle)
- Тянуть белую точку → изменить радиус
- Белая точка скрывается вместе с кругом

### Маршрутные линии
Линии между вейпоинтами — **касательные** к кругам (как ремень на шестерёнках).
Учитывается направление кружения (CW/CCW) для правильных касательных.

Дуги на кругах показывают путь дрона от входа до выхода:
- **CW** → дуга по часовой стрелке
- **CCW** → дуга против часовой

## 4. СПЕЦИАЛЬНЫЕ ТОЧКИ VTOL

### StartCircle (S) — жёлтый
| Свойство | Значение |
|----------|----------|
| Позиция | 300м от HOME на NE (45 градусов) |
| Радиус | 150м |
| Высота | 30м (= высота взлёта) |
| AutoNext | false (ждёт оператора) |
| Назначение | Кружение после взлёта и перехода в FW |

**Зачем S нужен**: После перехода из мультикоптерного в самолётный режим дрон набирает скорость и начинает кружить. Оператор визуально проверяет стабильность и даёт команду продолжить.

### LandingCircle (L) — оранжевый
| Свойство | Значение |
|----------|----------|
| Позиция | 300м от HOME на SW (225 градусов) |
| Радиус | 150м |
| Высота | 30м (= высота посадки) |
| AutoNext | false (ждёт оператора) |
| Назначение | Кружение перед переходом в MC и посадкой |

**Зачем L нужен**: Перед посадкой VTOL переходит обратно в мультикоптерный режим. Оператор подтверждает готовность площадки и условия для безопасного приземления.

### Управление S и L
- **Перетащить** мышкой → изменить позицию
- **Двойной клик** → диалог редактирования (высота, радиус, AutoNext, направление)
- Оба создаются автоматически при первом UpdateRoute() в режиме QuadPlane
- Отображаются с кругом радиуса и подписью ("СТАРТ" / "ПОСАДКА")
- Иконка "крутить" рядом с маркером = AutoNext выключен (дрон будет ждать)

## 5. УПРАВЛЕНИЕ МИССИЕЙ В ПОЛЁТЕ

### Кнопка "Заморозить / Продолжить миссию"

Одна кнопка внизу карты переключается между двумя состояниями:

#### Заморозить (синяя)
1. Проверяет что дрон Armed и подключен
2. Проверяет безопасность (seq > 2, не на посадке)
3. Переводит дрон в режим **LOITER** — кружит на месте
4. Кнопка становится зелёной "Продолжить миссию"

#### Продолжить (зелёная)
1. Переотправляет миссию (с возможными изменениями)
2. Устанавливает следующий WP (SetCurrentWaypoint(seq+1))
3. Переводит дрон в режим **AUTO**
4. Кнопка возвращается к "Заморозить"

### Защита от опасных действий
| Ситуация | Реакция |
|----------|---------|
| Дрон на взлёте (seq <= 2) | Нельзя заморозить: дрон взлетает |
| Дрон на посадке (TRANSITION_MC, LAND) | Нельзя заморозить: дрон на посадке |
| Дрон не Armed | Дрон не активирован |
| Нет подключения | Дрон не подключен |

### Реал-тайм редактирование
Пока дрон кружит на точке с AutoNext=false:
1. Оператор может перемещать, добавлять, удалять вейпоинты
2. Система автоматически переотправляет миссию при изменениях
3. Дрон продолжает кружить до явной команды "Продолжить"

## 6. НИЖНЯЯ ПАНЕЛЬ (ЛЕНТА МИССИИ)

Горизонтальная полоса внизу экрана показывает последовательность миссии в виде карточек.

### QuadPlane
```
МИССИЯ: [ВЗЛЁТ] → [FW] → [S СТАРТ] → [1 100м] → [2 120м] → [3 100м] → [L ПОСАДКА] → [MC] → [ЗЕМЛЯ]
```

### Copter/Plane
```
МИССИЯ: [H HOME] → [1 100м] → [2 120м] → [3 100м] → [RTL]
```

### Обозначения на карточках
- Символ кружения = AutoNext выключен (дрон ждёт)
- xN = LoiterTurns = N кругов перед продолжением
- число + "м" = высота вейпоинта

### Интерактивность
- **Клик** по карточке → карта центрируется на этом WP
- **Подсветка** — текущий WP дрона пульсирует (яркая рамка + анимация)
- **Прокрутка** — горизонтальный скролл для длинных миссий

## 7. СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### Сценарий A: Стандартная съёмка (QuadPlane)
```
1. Тип ЛА: QuadPlane
2. Расставить 8 WP по периметру объекта
3. Каждый WP: AutoNext=true, Turns=1, Radius=100м
4. WP4: AutoNext=false (съёмка крупного объекта)
5. Активировать миссию
6. Дрон: взлёт → FW → S (ждёт) → оператор "Продолжить"
7. Дрон летит WP1→WP3 автоматически
8. На WP4 кружит → оператор снимает камерой
9. Оператор нажимает "Продолжить" → WP5→WP8
10. Дрон → L (ждёт) → оператор "Продолжить" → MC → посадка
```

### Сценарий B: Изменение маршрута в полёте
```
1. Дрон летит к WP3
2. Оператор замечает: нужно облететь препятствие
3. Нажимает "Заморозить" → дрон кружит
4. Добавляет WP3a между WP2 и WP3
5. Сдвигает WP3 в сторону
6. Нажимает "Продолжить" → миссия обновлена
7. Дрон летит по новому маршруту
```

### Сценарий C: Мониторинг (Copter)
```
1. Тип ЛА: Copter
2. 5 точек над полем, все AutoNext=false
3. Активировать миссию
4. Дрон летит к WP1 → кружит
5. Оператор наблюдает с камеры, делает фото
6. "Продолжить" → WP2 → кружит → фото → ...
7. После WP5 → RTL → посадка
```

### Сценарий D: Аварийная ситуация
```
1. Дрон в автоматическом режиме
2. Видна опасная зона впереди
3. "Заморозить" → LOITER
4. Оценка ситуации
5. Если безопасно: "Продолжить"
6. Если опасно: нажать режим "RTL" в правой панели
```

## 8. ВОЗМОЖНЫЕ ОШИБКИ И РЕШЕНИЯ

### КРИТИЧЕСКИЕ

| Проблема | Причина | Решение |
|----------|---------|---------|
| Дрон не переходит к следующему WP | AutoNext=false, оператор не нажал "Продолжить" | Нажать "Продолжить" или перевести в AUTO |
| Дрон кружит бесконечно после потери связи | LOITER_UNLIM + нет failsafe | Настроить FS_LONG_ACTN=1 в ArduPilot |
| Дрон не может вписаться в радиус (VTOL) | Радиус меньше мин. разворота FW | Увеличить радиус больше 80м |
| Сваливание при заморозке на переходе | Попытка LOITER при переходе FW-MC | Защита: блокирует на seq<=2 и при посадке |
| Миссия не загружается | Слишком много точек / таймаут | Уменьшить количество WP, проверить связь |

### СРЕДНИЕ

| Проблема | Причина | Решение |
|----------|---------|---------|
| S/L не видны на карте | Тип ЛА не QuadPlane | Переключить на QuadPlane |
| Радиус не изменяется | Клик не попал в resize handle | Кликнуть точно по белой точке на краю круга |
| Дуга пересекает маршрут | Круги перекрываются | Раздвинуть WP или уменьшить радиусы |

## 9. ЧЕКЛИСТ ПЕРЕД ВЫЛЕТОМ

### Обязательно
- [ ] GPS зафиксировано, HOME установлен корректно
- [ ] Тип ЛА соответствует реальному
- [ ] Все WP на безопасной высоте (20м+ для Copter, 30м+ для VTOL)
- [ ] Радиусы кружения 80м+ для QuadPlane
- [ ] AutoNext настроен на нужных точках
- [ ] Маршрут не проходит через запретные зоны
- [ ] Батарея достаточна для маршрута + RTL

### QuadPlane дополнительно
- [ ] S и L видны на карте, не перекрываются с WP
- [ ] S расположен на достаточном расстоянии от HOME для разгона
- [ ] L расположен удобно для захода на посадку
- [ ] Высота S и L >= 30м

### ArduPilot настройки (рекомендуемые)
```
FS_LONG_ACTN = 1        (RTL при потере связи)
FS_LONG_TIMEOUT = 20    (таймаут секунд)
Q_RTL_MODE = 1          (VTOL RTL через MC)
Q_OPTIONS = 16          (переход в любом режиме)
WPNAV_RADIUS = 200      (радиус достижения WP, см)
RTL_ALT = 3000           (высота RTL, см)
```

## 10. АРХИТЕКТУРА MAVLINK

### Последовательность VTOL миссии
```
seq=0:   HOME           (cmd=16, NAV_WAYPOINT)
seq=1:   VTOL_TAKEOFF   (cmd=84, z=высота)
seq=2:   TRANSITION_FW  (cmd=3000, param1=4)
seq=3:   START_CIRCLE   (cmd=17 или cmd=18)
seq=4:   WP1            (cmd=16/17/18)
seq=5:   WP2
...
seq=N+3: WPN
seq=N+4: LANDING_CIRCLE (cmd=17/18)
seq=N+5: TRANSITION_MC  (cmd=3000, param1=3)
seq=N+6: VTOL_LAND      (cmd=85, координаты HOME)
```

### Последовательность Copter/Plane
```
seq=0:   HOME           (cmd=16)
seq=1:   WP1            (cmd=16/17/18)
seq=2:   WP2
...
seq=N:   WPN
```
RTL обрабатывается через режим полёта, не через миссию.

### MAV_CMD коды
| Код | Название | Использование |
|-----|----------|---------------|
| 16 | NAV_WAYPOINT | Пролёт точки (param1=delay) |
| 17 | LOITER_UNLIM | Бесконечное кружение (param3=radius) |
| 18 | LOITER_TURNS | N кругов (param1=turns, param3=radius) |
| 84 | VTOL_TAKEOFF | Вертикальный взлёт (z=altitude) |
| 85 | VTOL_LAND | Вертикальная посадка |
| 3000 | DO_VTOL_TRANSITION | Переход: param1=4(FW), param1=3(MC) |

### Знак радиуса в MAVLink
- **Положительный** radius → CW (по часовой)
- **Отрицательный** radius → CCW (против часовой)
- Это стандарт ArduPilot для LOITER команд
